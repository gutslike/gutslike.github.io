# Welcome!

I will be following the Python roadmap from **roadmap.sh** as a structured guide throughout this series.

> Roadmap: https://roadmap.sh/python

Although I already know the basics of Python and have used it in several projects, I'm starting from the beginning for the sake of revision. This time, however, my goal is to understand every concept more deeply instead of just learning enough to use it.

Once I have covered the roadmap, I plan to start solving **LeetCode** and **Codeforces** problems using Python to strengthen my problem-solving skills and gain a better understanding of the language. Those solutions, along with explanations and approaches, will be documented in a separate series on this blog.

## What to expect from this series

This series is intended to be a **concise revision guide**, not a complete Python textbook.

Rather than explaining every topic in exhaustive detail, I'll focus on:

- The core concepts you need to understand.
- Important syntax and examples.
- Common pitfalls and best practices.
- Quick explanations for efficient revision.
- Useful references for deeper learning when needed.

My aim is to create notes that are useful not only for readers but also for myself whenever I need a quick refresher on a topic.

If you're following the same roadmap or preparing for interviews, competitive programming, or Python-based development, I hope you'll find this series useful.
